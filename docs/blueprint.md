# **App Name**: UniVerse

## Core Features:

- Skill-Based Matching: Connect students based on skills, interests, and project preferences to facilitate team formation.
- AI-Powered Project Idea Generation: Use a tool which analyzes student profiles and generates personalized project ideas and relevant resources.
- Real-Time Collaboration: Enable students to collaborate on projects with real-time code editing, document sharing, and instant messaging.
- Event Discovery: Aggregate hackathons, workshops, and student events, with personalized recommendations and registration links.
- Community Forums: Enable discussions on various topics, encouraging students to share insights, ask questions, and engage in peer learning.
- Personalized Dashboard: Create a personalized dashboard for each student, with project updates, community activity, and suggested teammates.
- Reputation System: Reward student contributions with reputation badges and recognition to foster engagement and incentivize collaboration.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to capture a cosmic, techy aesthetic.
- Background color: Dark navy (#1A237E) with low saturation for a futuristic feel.
- Accent color: Electric pink (#F06292) to create high contrast and visual interest, calling attention to CTAs.
- Headline font: 'Inter', a sans-serif with a modern, neutral feel.
- Body font: 'Plus Jakarta Sans', as requested.
- Use minimal line icons in the Lucide style, with strokes of accent colors for interactive elements.
- Layout is inspired by daily.dev and Reddit, focusing on content cards, clear sections, and sidebars for navigation and recommendations.
- Implement subtle animations and transitions to enhance user experience, such as hover elevation on cards.
